Set Up a Project First

Stop learning prompts. Projects are the real secret — and most tutorials skip this.

Scattered chats forget everything. A project remembers — and gets smarter each time.

1

What is a project?

A project is like an organizing folder with memory. Instead of starting fresh every chat, the AI remembers your preferences, files, and past conversations. It's the single biggest upgrade most people never make.

2

The Rule of Two

If you do something more than twice — weekly reports, customer emails, lesson plans — make it a project. That's it. If it's recurring, it deserves a project.

3

How to create one

Claude: Click "Create Project" in the sidebar.
ChatGPT: Click the project icon in the sidebar. Name it after what you're working on — "Weekly Reports," "Client Emails," "Course Content."

💡
Pro tip: Once your project is created, ask the AI to write its own instructions: "Help me write the best custom instructions for this project." It knows what it needs better than you do.
